# API-React-native

Implementação de Aplicativo Mobile (React/TSX)

Este projeto consiste na implementação de um aplicativo frontend desenvolvido em React JS (TSX), simulando uma experiência de usuário mobile (Mobile-First Design) com o framework Tailwind CSS.

Funcionalidades e Escopo

- Consumo de API: O aplicativo se conecta a uma API pública (ghibliapi.vercel.app/films) para buscar e exibir uma lista de longas-metragens do Studio Ghibli.

Visualização e Navegação:

- Lista Principal: Exibe uma lista de filmes com posters, títulos e informações resumidas.

- Filtragem: Possui um campo de busca que permite filtrar a lista por título em tempo real.

Tela de Detalhes: Implementa uma navegação em Single-Page Application (SPA), onde o clique em qualquer item da lista leva o usuário a uma tela dedicada que exibe a sinopse completa, diretor, data de lançamento e poster do filme selecionado.
